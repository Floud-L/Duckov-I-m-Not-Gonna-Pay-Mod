# 逃离鸭科夫MOD - 我是不会还钱的! (I'm Not Gonna Pay!)

**Steam创意工坊链接:**  
https://steamcommunity.com/sharedfiles/filedetails/?id=3593492783

## 安装教程

### 步骤 1: 下载MOD文件
前往Release页面下载最新版的mod压缩包

### 步骤 2: 打开游戏根目录
找到你的《逃离鸭科夫》游戏安装目录

### 步骤 3: 安装MOD文件
将下载的压缩包解压后，把整个mod文件夹放入以下路径：
```
Escape from Duckov\Duckov_Data\Mods\
```
(如果Mods文件夹不存在，请手动新建一个)

### 正确的文件结构应该是：
```
Escape from Duckov/
├── Duckov_Data/
│   ├── Mods/
│   │   ├── ImNotGonnaPay/
│   │   │   ├── Bank.dll
│   │   │   └── ... (其他文件)
```

### 步骤 4: 启用MOD
1. 启动游戏
2. 进入游戏内的mod管理器
3. 找到"I'm Not Gonna Pay"并启用它
