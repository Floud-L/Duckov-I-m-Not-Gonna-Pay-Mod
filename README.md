逃离鸭科夫MOD  我是不会还钱的! - I'm Not Gonna Pay!
steam链接:
https://steamcommunity.com/sharedfiles/filedetails/?id=3593492783

安装教程:
1.去Release下载最新版的mod压缩包
2.打开游戏根目录
3.将文件解压后放入:Escape from Duckov\Duckov_Data\Mods目录(如果没有Mods文件夹则新建一个)
你的文件结构应该是:
Escape from Duckov:
  Duckov_Data:
    Mods:
      ImNotGonnaPay:
        Bank.dll
4.打开游戏并在mod管理器中启用mod
